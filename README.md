# AppCommon
App的基础类库 (持续更新)

##### notice : all classes pre suffix is 'PhXxxxx' for search all class  

# 引入项目

#### Gradle dependency:

```
compile 'com.lijiangping:appcommon:1.0.1'
```


#### Maven dependency:

```
<dependency>
  <groupId>com.lijiangping</groupId>
  <artifactId>appcommon</artifactId>
  <version>1.0.1</version>
  <type>pom</type>
</dependency>
```
